# SportNest Client

Next.js frontend for SportNest – Sports Facility Booking System.

## Setup

```bash
npm install
```

Create `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

## Run

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Development server |
| `npm run build` | Production build |
| `npm run start` | Production server |

## Deploy

Deploy this folder to Vercel or Netlify. Set `NEXT_PUBLIC_API_URL` to your live Express API URL.
